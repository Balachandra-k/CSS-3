CSS-3 RADIAL-GRADIENTS

In a redial color emerge from a single point and smoothly spreed outward in a circle to ellipitical rather than fading from one color to another in a single direction as with liner gradients. the basic syontax or creating a redical can be given with

SYNTAX : RADIAL-GRADIENTS(shape size at position,color-stopl1 color-stop2,...)

The arugments of the RADIAL-GRADIENTS()function has the following meaning:

  1. Position- spicifies the starting point of the gradient,wich can be specified in units (px,em,or percentages) or keyword (left,button ,etc..
  2. Shape- specifies the shape of the gradien's ending shape. it can be circle or ellipse
  3. Size-specifies the size of the gradien's ending shape the diffacult its farther-side
